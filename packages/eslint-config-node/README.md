# eslint-config-node

## Project setup
```
yarn add @openeagle/eslint-config-node -D
```

### Compiles and hot-reloads for development
1、项目根目录爱下 新建 .eslint.js 文件，添加以下代码
```js
module.exports = {
  extends: ['@openeagle/eslint-config-node'],
}
```
